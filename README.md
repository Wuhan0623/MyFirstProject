# README

* **项目名称**
> 词频统计个人作业

* **运行方式**
> 通过命令行依次输入两个参数，即要统计的文本文件路径和统计结果输出的文本文件路径。

* **功能简介**
> 1.统计文件的字符数（对应输出第一行）
> 
> 2.统计文件内的至少以4个英文字母开头的单词总数（对应输出第二行）
>
> 3.统计文件的有效（任何包含非空白字符的行）行数（对应输出第三行）
>
> 4.统计文件中各单词的出现次数（对应输出接下来10行），最终只输出频率最高的10个
